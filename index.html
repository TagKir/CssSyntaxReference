CSS Syntax Reference

Basic syntaxs

Syntax Example Description
* * Selects all elements
type h1 Selects the type of elements
.class .example_class The base syntax selects elements based on the value of the class attribute
#id #12345 Identifies an element by its unique identifier
type[attr]|[attr] a[href="..."] Selects all elements that have this attribute or attribute with a certain value

Combinators

Syntax Example Description
A, B div, span Selects all elements
A B div span Selects the elements that are inside the specified element
A > B ul li Selects only those elements that are child elements directly in relation to the specified element
A ~ B p ~ span Selects items that are at the same nesting level after the specified element, with the same parent
A + B h2 + p Selects an element that is immediately after the specified element if they have a common parent

Pseudo-classes and Pseudo-elements

Syntax Example Description
:pseudo a:visited Allows you to select items based on information that is not in the item tree
::pseudo p::first-line Allows you to select things that are not in HTML
